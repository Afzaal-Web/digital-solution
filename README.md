## Sections

1. **Header**: Fixed navigation with company logo and menu
2. **Hero Slider**: Full-width background images with overlay text
3. **Services**: Grid of service cards with icons
4. **Portfolio**: Image gallery with hover effects and modal popups
5. **About**: Company information with statistics
6. **Contact**: Contact form and company information
7. **Footer**: Links, social media, and copyright

## Usage

1. Open `index.html` in any modern web browser
2. The website is fully functional and ready to use
3. All images are loaded from royalty-free sources (Unsplash)
4. No additional setup or dependencies required

## Customization

### Colors
The main brand color is `#2563eb` (blue). You can change this throughout the CSS file to match your brand.

### Content
- Update company name in the navigation and footer
- Replace placeholder text with your actual content
- Add your own images by replacing the Unsplash URLs
- Modify contact information in the contact section

### Images
All images are currently sourced from Unsplash. To use your own images:
1. Replace the `src` attributes in the HTML
2. Ensure images are optimized for web use
3. Maintain aspect ratios for best results
